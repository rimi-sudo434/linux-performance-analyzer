## Overview
# Linux Performance Analyzer



A lightweight and beginner-friendly Linux system monitoring tool written in Python. This command-line application collects and displays essential system information, making it useful for learning Linux system programming and basic performance monitoring.



---



## Overview



Linux Performance Analyzer is designed to provide a quick overview of your Linux system directly from the terminal. It retrieves important system details such as operating system information, hostname, current user, disk usage, and the current date and time.



This project is intended for students, Linux beginners, and developers who want to understand how Python can interact with the Linux operating system.



---



## Features



- Displays Operating System information

- Displays Hostname

- Displays Current User

- Shows Total Disk Space

- Shows Used Disk Space

- Shows Free Disk Space

- Displays Current Date and Time

- Lightweight command-line interface

- Easy to understand source code

- Beginner-friendly project structure

## Installation



Clone the repository



```bash

git clone https://github.com/rimi-sudo434/linux-performance-analyzer.git

```



Move into the project directory



```bash

cd linux-performance-analyzer

```



Run the application



```bash

python3 analyzer.py

```
